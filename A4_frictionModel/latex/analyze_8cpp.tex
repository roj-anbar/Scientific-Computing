\doxysubsection{Detailed Description}
Analyzes a sample spatial timeseries dataset to estimate the friciton rate. 

\begin{DoxyAuthor}{Author}
Ramses van Zon (commented by Rojin Anbarafshan) 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
February 13, 2025 
\end{DoxyDate}
This program reads a sample timeseries position dataset from a file, computes velocity using numerical differentiation, estimates the friction rate, and finally writes the results to an ouput file.

The script supports command-\/line arguments to specify input and output files.
